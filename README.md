## Deploy

```bash
# Clone and deploy to Vercel
git clone https://github.com/your-username/stellarexpert
cd stellarexpert
vercel
```

The project is a single `index.html` file — no build step required.

---

## Stack

- Plain HTML, CSS, JavaScript — no framework
- [Tabler Icons](https://tabler.io/icons) for iconography
- [Syne](https://fonts.google.com/specimen/Syne) + [IBM Plex Mono](https://fonts.google.com/specimen/IBM+Plex+Mono) via Google Fonts
- Hosted on [Vercel](https://vercel.com)
